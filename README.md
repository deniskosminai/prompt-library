# prompt-library
# 📚 Библиотека Промптов (LLMOps)

Это  репозиторий для хранения, версионирования и автоматического тестирования промптов.

## 🛠 Структура:
- `/prompts/` — шаблоны промптов разделенные по папкам (одна задача = одна папка).
- `/tests/` — автотесты `promptfoo` и JSON датасеты.
- `/.github/workflows/` — CI/CD пайплайны.

## 🚀 Как запустить тесты локально
1. Установите инструмент: `npm install -g promptfoo`
2. Перейдите в папку тестов: `cd tests`
3. Запустите: `npx promptfoo eval` (потребуется ключ API в переменных окружения).
